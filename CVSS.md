
# Common Vulnerability Scoring System (CVSS)

| Severity  | Base Score Range |
|---|---|
| **None** | 0.0 |
| **Low** | 0.1-3.9 |
| **Medium** | 4.0-6.9|
|**High**| 7.0-8.9 |
|**Critical**| 7.0-10|

[National Vulnerability Database (NVD)](https://nvd.nist.gov)
